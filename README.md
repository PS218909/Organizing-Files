# Organizing-Files
 Organizing files based on their extensions

Let their be a directory x containing asd.txt, lkhg.txt, qwer.csv, psd.docx
Running this file on that directory will make changes as given below: -
```
x
    txt
        asd.txt
        lkhg.txt
    csv
        qwer.csv
    docx
        psd.docx
```
